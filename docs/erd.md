## Users

- id (ObjectId, PK)
- name (String)
- email (String, unique)
- password (String)
- avatar (String)
- bio (String)
- skills (Array of Strings)
- githubUsername (String)
- role (String: "user" | "admin")
- voided (Boolean)
- voidedReason (String)
- voidedAt (Date)
- voidedBy (ObjectId, FK -> Users.id)
- createdAt (Date)
- updatedAt (Date)

## Projects

- id (ObjectId, PK)
- userId (ObjectId, FK -> Users.id)
- title (String)
- description (String)
- techStack (Array of Strings)
- githubLink (String)
- demoLink (String)
- screenshots (Array of Strings)
- category (String: e.g., "Web Dev", "AI")
- likes (Array of ObjectIds -> Users.id)
- comments (Array of Embedded Documents)
- voided (Boolean)
- voidedReason (String)
- voidedAt (Date)
- voidedBy (ObjectId, FK -> Users.id)
- createdAt (Date)
- updatedAt (Date)
